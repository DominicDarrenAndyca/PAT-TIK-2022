There are 12 skills in Hypixel Skyblock. Those skills are farming, mining, combat, foraging, fishing, enchanting, alchemy, taming, dungeoneering, and 3 skills that nobody cares about: runecrafting, carpentry, and social.

Leveling up skills requires a certain amount of skill xp of that skill (Example: The farming skill requires 150 farming xp to level up

Some skills can be easy to level up, like enchanting and alchemy. But some skills are hell to level up, like foraging

The maximum level of each skill is 60 for Farming, Mining, Enchanting and Combat, 50 for Foraging, Fishing, Taming, Brewing Stand.png Alchemy, Dungeoneering and Carpentry, and 25 for Social and Runecrafting.
